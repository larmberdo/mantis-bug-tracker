# Mantis ECMS Bug 跟踪记录

这是一个用于跟踪和管理 Mantis ECMS 系统 Bug 的静态网页应用。

## 🌐 在线访问

访问地址：`https://larmberdo.github.io/mantis-bug-tracker`

## 📋 功能特性

- 📊 直观的Bug状态展示
- 🎨 现代化的UI设计
- 📱 响应式布局，支持移动端
- 🔍 清晰的Bug分类和状态标识
- 📅 按日期组织的Bug记录
- 🟡 新增"已修复，待验证"状态
- 🔽 高级筛选功能：按状态、日期、处理人员筛选
- 📈 实时统计概览，显示各状态Bug数量

## 🚀 部署说明

本项目使用 GitHub Pages 进行部署，每次推送到 `main` 分支后会自动更新网站内容。
